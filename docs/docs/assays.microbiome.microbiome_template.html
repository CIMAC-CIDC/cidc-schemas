Metadata file for microbiome.

Description: Metadata information for Microbiome data submission.

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
assay creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
batch id Microbiome batch identification number. < string >
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
forward fastq Information about a Gzipped FASTQ file. fastq.gz
forward index Information about a Gzipped FASTQ file. fastq.gz
reverse fastq Information about a Gzipped FASTQ file. fastq.gz
reverse index Information about a Gzipped FASTQ file. fastq.gz
otu table Information about a tab Separated file. tsv

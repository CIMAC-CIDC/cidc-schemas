CIDC Documentation

RNA expression [BAM] assay template

Description: RNA expression data submission via BAM files.

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
enrichment method Method used for enriching for mRNA. assays.rna_assay-v0#enrichment_method
enrichment kit Vendor for the bait set used for enrichment, e.g. Twist, Agilent, IDT. assays.rna_assay-v0#enrichment_vendor_kit
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. assays.components.ngs_assay_core#sequencer_platform
paired end reads Indicates if the sequencing was performed paired or single ended. assays.components.ngs_assay_core#paired_end_reads

Samples
Property Description Data Type
cimac id Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. sample#cimac_id
BAM file A comma-separated list of paths to files on a user's computer. assays.components.multiple_local_files#file_path_list
library yield ng Resulting yield (in ng) from library construction. assays.rna_assay-v0#records.items.library_yield_ng
dv200 The percentage of fragments > 200 nucleotides. (0-1) assays.rna_assay-v0#records.items.dv200
rqs RNA quality score. (0-10) assays.rna_assay-v0#records.items.rqs
rin RNA integrity number. (0-10) assays.rna_assay-v0#records.items.rin
quality flag Flag used for quality. < number >

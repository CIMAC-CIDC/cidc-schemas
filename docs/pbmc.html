CIDC Data Model

Shipping manifest for PBMC samples

Description: Shipping / receiving manifest for transferring PBMC samples.

Core Columns: Manifest Header
ID Description Data Type
clinical_trial.lead_organization_study_id Trial identifier used by lead organization, e.g. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. <string>
shipping_core.manifest_id Filename of the manifest used to ship this sample. Example: E4412_PBMC. <string>
shipping_core.request Assigns a request identifier (mandatory or optional) for the assay. <string>
shipping_core.assay_priority Priority of the assay as it appears on the intake form. <string>
  • Assay-1
  • Assay-2
  • Assay-3
  • Assay-4
  • Assay-5
  • Assay-6
  • Assay-7
  • Assay-8
  • Assay-9
  • Assay-10
  • Assay-11
  • Assay-12
  • Assay-13
  • Assay-14
  • Assay-15
  • Other
shipping_core.assay_type Assay and sample type used. Example: Green-top tube - Plasma (Olink). <string>
shipping_core.batch_number The number corresponding to the batch sent for this assay. <integer>
shipping_core.box_number Box identifier for samples, if sample shipment container includes multiple boxes for each sample/assay type. <integer>
shipping_core.assay_run_number Assay run number for samples received. <integer>
shipping_core.courier Courier utilized for shipment. <string>
  • FEDEX
  • USPS
  • UPC
shipping_core.tracking_number Air bill number assigned to shipment. Example: 4567788343. <string>
shipping_core.account_number Courier account number to pay for shipping if avaialable. Example: 45465732. <string>
shipping_core.shipping_condition Type of shippment made. <string>
  • Frozen (Dry Ice)
  • Frozen (Dry Shipper)
  • Ambient (Ice Pack)
  • Ambient
  • Other
shipping_core.date_shipped Date of shipment. <string> date-time
shipping_core.num_of_samples_shipped Count of specimens distributed by site. <integer>
shipping_core.num_of_samples_good Number of Samples Received in Good Condition. <integer>
shipping_core.num_of_path_report Number if pathology report received. <integer>
shipping_core.ship_from Shipping from [free text: person or organization, address, phone number, email address.] <string>
shipping_core.ship_to Shipping to [free text: person or organization, address, phone number, email address.] <string>
Shipping Columns: Completed by the BioBank
ID Description Data Type
clinical_trial.lead_organization_study_id Trial identifier used by lead organization, e.g. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. <string>
participant.trial_participant_id Trial Participant Identifier. Example: PT123. <string>
participant.cimac_participant_id Participant identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-1234. <string>
participant.cohort_id Identifies the treatment cohort to which the participant belongs. Example: Cohort A. <string>
participant.arm_id Identifies the trial arm to which the participant belongs. Example: Arm A. <string>
sample.site_sample_id Sample identifier assigned by the biorepository site. Example: EA123-45. <string>
sample.cimac_sample_id Specimen identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. <string>
sample.time_point Study related collection time point, as described in the protocol intake form. Example: Progression. <string>
sample.blood_collection_date Date of blood collection. <string> date
sample.blood_collection_time Time of blood collection. <string> time
sample.blood_processing_date Date of blood processing. <string> date
sample.blood_processing_time Time of blood processing. <string> time
sample.sample_location Sample location within the shipping container. Example: A1. <string>
sample.specimen_type Type of sample sent. <string>
  • Core Needle Biopsy
  • Surgical Resection
  • Endoscopic Biopsy
  • Punch Biopsy
  • Plasma
  • PBMC
  • Whole Blood
  • Stool Sample
  • Bone Marrow Aspirate
  • Other
sample.specimen_format The format in which the sample was sent. <string>
  • FFPE Block
  • FFPE Scroll
  • FFPE Core Punch
  • Unstained Slide
  • Frozen Tissue
  • Sodium Heparin Green-Top Tube
  • Streck Tube
  • EDTA Tube
  • Archival FFPE Block
  • Archival FFPE Slides
  • Archival FFPE Core Punch
  • Archival FFPE Scroll
  • Other
  • Frozen Stool
  • DNA-Preserved Stool
  • Other
aliquot.quantity Quantity of each sample shipped. <integer>
aliquot.volume_quantity Volume of sample shipped. <number>
aliquot.volume_units Volume units. Example: ml. <string>
aliquot.pbmc_aliquoted Site or biobank provides the final number of PBMCs per vial that were frozen. <number>
aliquot.sample_source Indicates the specimen source of the sample shipped. Example: Na Heparin blood draw aliquots (2 of three), FFPE block #52 <string>
Receiving Columns: Completed by the CIMAC
ID Description Data Type
aliquot.cimac_aliquot_id Aliquot identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. <string>
aliquot.pbmc_recovered Receiving site determines number fo PBMCs per vial recovered upon receipt. <number>
aliquot.material_used Receiving site indicates how much material was used for assay purposes. <number>
aliquot.material_remaining Receiving site indicates how much material remains after assay use. <number>
aliquot.sample_quality_status Final status of sample after QC and pathology review. <string>
  • Usable for Assay
  • Not Usable for Assay
  • Verifying
  • Other
aliquot.sample_replacement Status of sample if replacement is/was requested. <string>
  • Replacement Requested
  • Replacement Tested
  • N/A
aliquot.sample_status Status of sample used for other assay, exhausted, destroyed, or returned. <string>
  • Sample Returned
  • Sample Exhausted
  • Remainder used for other Assay
  • Other

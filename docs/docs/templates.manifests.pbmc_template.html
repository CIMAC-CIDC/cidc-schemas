CIDC Data Model

Shipping manifest for PBMC samples

Description: Shipping / receiving manifest for transferring PBMC samples.

Worksheet - Shipment

Preamble Rows

Property Description Data Type
protocol id Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_id
manifest id Filename of the manifest used to ship this sample. Example: E4412_PBMC. shipping_core#manifest_id
request Assigns a request identifier (mandatory or optional) for the assay. shipping_core#request
assay priority Priority of the assay as it appears on the intake form. shipping_core#assay_priority
assay type Assay and sample type used. Example: Green-top tube - Plasma (Olink). shipping_core#assay_type
batch number The number corresponding to the batch sent for this assay. shipping_core#batch_number
box number Box identifier for samples, if sample shipment container includes multiple boxes for each sample/assay type. shipping_core#box_number
assay run number Assay run number for samples received. shipping_core#assay_run_number
courier Courier utilized for shipment. shipping_core#courier
tracking number Air bill number assigned to shipment. Example: 4567788343. shipping_core#tracking_number
account number Courier account number to pay for shipping if available. Example: 45465732. shipping_core#account_number
shipping condition Type of shipment made. shipping_core#shipping_condition
date shipped Date of shipment. shipping_core#date_shipped
date recieved Date of receipt. shipping_core#date_recieved
quality of shipment Final status of aliquot after QC and pathology review. shipping_core#quality_of_shipment
num of samples shipped Count of specimen(s) distributed by site. shipping_core#num_of_samples_shipped
num of samples good Number of Sample(s) Received in Good Condition. shipping_core#num_of_samples_good
num of path report Number of pathology report(s) received. shipping_core#num_of_path_report
ship from Shipping from [free text: person or organization, address, phone number, email address.] shipping_core#ship_from
ship to Shipping to [free text: person or organization, address, phone number, email address.] shipping_core#ship_to

Worksheet - Samples

Data Columns

Filled by Biorepository
Property Description Data Type
participant id Trial Participant Identifier. Example: PT123. participant#participant_id
cohort name Categorical description of cohorts, arms, and treatment groups. Example: Cohort A. participant#cohort_name
arm id Identifies the trial arm to which the participant belongs. Example: Arm A. participant#arm_id
parent sample id Sample identifier assigned by the biorepository site. Example: EA123-45. sample#parent_sample_id
processed sample id Aliquot identifier assigned by the biorepository site: EA123-45-1 sample#processed_sample_id
cimac id Specimen identifier assigned by the CIMAC-CIDC Network. Formated as CM-????-????-?? sample#cimac_id
collection event name Categorical description of timepoint at which the sample was taken. sample#collection_event_name
genomic source The type of material submitted - tumor/normal/etc sample#genomic_source
blood collection date Date of blood collection. sample#blood_collection_date
blood collection time Time of blood collection. sample#blood_collection_time
blood processing date Date of blood processing. sample#blood_processing_date
blood processing time Time of blood processing. sample#blood_processing_time
sample processing compliance Records if a sample processing (blood or tissue) is not compliant with protocol. sample#sample_processing_compliance
sample location Sample location within the shipping container. Example: A1. sample#sample_location
type of sample Type of sample sent. sample#type_of_sample
sample collection procedure Indicates the specimen source of the sample shipped. Example: Na Heparin blood draw aliquots (2 of three), FFPE block #52 sample#sample_collection_procedure
type of primary container The format in which the sample was sent. sample#type_of_primary_container
quantity Quantity of each aliquot shipped. aliquot#quantity
sample volume Volume of aliquot shipped. sample#sample_volume
sample volume units The unit of biological material from this sample. sample#sample_volume_units
Filled by CIMAC Lab
Property Description Data Type
pbmc viability Receiving site determines the percent recovered cells that are viable after thawing. sample#pbmc_viability
pbmc recovery Receiving site determines number for PBMCs per vial recovered upon receipt. sample#pbmc_recovery
pbmc resting period used Receiving site indicates if a resting period was used after PBMC recovery. sample#pbmc_resting_period_used
material used Receiving site indicates how much material was used for assay purposes. sample#material_used
material remaining Receiving site indicates how much material remains after assay use. sample#material_remaining
sample replacement Indication if sample replacement is/was requested. sample#sample_replacement
residual sample use Indication if sample was sent to another location or returned back to biorepository. sample#residual_sample_use
comments Comments on sample testing. sample#comments

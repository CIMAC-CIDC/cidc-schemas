IHC assay template

Description: ImmunoHistoChemistry (IHC) submission.

Property Description Data Type
protocol identifier
required 		Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
assay creator
required 		Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
slide scanner model
required 		Model version of the slide scanner instrument. < string >
  • "Vectra 2.0"
  • "Hamamatsu"
  • "AT2 Turbo"
staining platform
required 		Staining platform used for tissue sample labeling. < string >
  • "auto"
  • "manual"
autostainer model
required 		Model version of the autostainer instrument. < string >
  • "Bond RX"
  • "Bond MAX"
  • "Other"
  • "N/A"
antibody name
required 		Antibody type collected for this study. < string >
company
required 		Company from which antibody sample was derived. < string >
clone
required 		Resulting type of antibody clone from primary antibody sample. < string >
catalogue number
required 		Antibody registry catalog number assigned to antibody sample. < string >
lot number
required 		Identification number assigned to the particular quantity or lot of material from manufacturer. < string >
dilution
required 		Concentration ratio for antibody dilution. < string >
incubation time
required 		Time duration of antibody incubation. < string : time >
incubation temperature
required 		Temperature used for antibody incubation. < string >

^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
IHC image
required 		Information about an image file. image
Marker call
Property Description Data Type
marker positive
required 		Indicates whether the marker is considered positive by clinical trial guidelines (if applicable). < string >
  • "positive"
  • "negative"
  • "no_call"
Chromogenic IHC (please specify one score/metric)
Property Description Data Type
tumor proportion score Tumor Proportion Score (TPS) is the percentage of viable tumor cells showing marker staining relative to all viable tumor cells. (0-1) < number , string >
combined positive score Combined Positive Score (CPS) is the percentage of marker staining cells (tumor cells and cells that are non-tumor) relative to all viable tumor cells. (0-1) < number , string >
inflammatory cells Percentage of inflammatory cells (non-tumor cells) showing marker staining relative to all inflammatory cells. (0-1) < number , string >
positive inflammatory cell area Area of PD-L1+ Inflammatory Cells over the area of TSI + IT as a percentage. (0-1) < number , string >
intensity A measure of the intensity or brightness of the protein. (0-3) < number , string >
% expression A percentage of the relevant cells considered positive. (0-100) < number , string >
h score A summation of the percentage of area stained at each intensity level multiplied by the weighted intensity. (0-300) < integer , string >
comment A text comment regarding this slide. < string >

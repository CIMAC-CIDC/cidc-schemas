Description: Information about a FCS file.

Properties

Property Description Data Type
data_format Data format. < const: FCS >
upload_placeholder A placeholder for when artifact file is being uploaded. artifacts.artifact_core#upload_placeholder
artifact_creator The name of the center that created this artifact. artifacts.artifact_core#artifact_creator
uploader The name of the person uploading the artifact. artifacts.artifact_core#uploader
uuid UUID of artifact. artifacts.artifact_core#uuid
file_name The name of the file with extension. artifacts.artifact_core#file_name
object_url URL to artifact within Google Bucket. artifacts.artifact_core#object_url
uploaded_timestamp Timestamp of when artifact was loaded into the system. artifacts.artifact_core#uploaded_timestamp
file_size_bytes File size in bytes. artifacts.artifact_core#file_size_bytes
md5_hash MD5 Hash of artifact. Not available for composite GCS objects. artifacts.artifact_core#md5_hash
crc32c_hash CRC32c Hash of artifact. artifacts.artifact_core#crc32c_hash
visible Indicates if the artifact is visible. If set to false, the artifact is effectively deleted. artifacts.artifact_core#visible
artifact_category Artifact category. artifacts.artifact_core#artifact_category
facet_group The internal data category for this artifact artifacts.artifact_core#facet_group

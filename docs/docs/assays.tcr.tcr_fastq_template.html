CIDC Documentation

TCRseq [FASTQ] assay template

Description: TCR expression data submission via FASTQ files.

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
assay creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
batch id TCRseq batch identification number. < string >
sequencing run date Date when the TCR sequencing was run. < string : date >
sample sheet A Comma-Separated Value (CSV) text file describing the samples in this batch. csv

Replicates
Property Description Data Type
cimac id Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. < string > as regex
^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
replicate id Replicate identifier. < string >
read 1 Fastq file for the first fragment. fastq gz
read 2 Fastq file for the second fragment. fastq gz
index 1 Fastq file for the index of Read 1. fastq gz
index 2 Fastq file for the index of Read 2. fastq gz
rna quantity RNA quantity (in ng). < number >

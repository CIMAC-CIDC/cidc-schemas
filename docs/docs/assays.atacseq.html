ATACseq Assay

Description: Assay for Transposase-Accessible Chromatin by Sequencing

Property Description Data Type
assay_creator
required 		Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
batch_id
required 		ATACseq batch identification number. Unique to every batch. < string >
paired_end_reads
required 		Indicates if the sequencing was performed paired or single ended. < string >
  • "Paired"
  • "Single"
protocol_identifier
required 		Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
read_length
required 		Number of cycles for each sequencing read. < integer >
sequencer_platform
required 		Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
bait_set Bait set ID. < string >
library_kit The library construction kit. < string >
sequencing_protocol Protocol and version used for the sequencing. < string >

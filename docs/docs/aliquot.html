Aliquot

Description: Pertaining to a portion (volume or weight) of the whole.

Properties

Property Description Data Type

required 		Receiving site indicates how much material remains after assay use. < number >

  • Other

  • Replacement Requested
  • Replacement Tested
  • N/A

  • Aliquot Returned
  • Aliquot Exhausted
  • Remainder used for other Assay
  • Other
  • Aliquot Leftover

  • µL

  • cells/ml
  • N/A
  • Other

CYTOF Assay

Description: Information about the acquisition and analysis CyTOF data.

Property Description Data Type
assay_creator
required 		Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
assay_run_id
required 		User defined unique identifier for this assay run. Not necessarily unique across batches. < string >
batch_id
required 		CyTOF batch identification number. Unique to every batch. < string >
acquisition_buffer Cell staining buffer used for antibody and cell dilution. < string >
astrolabe_analysis Information about a ZIP file. zip
astrolabe_reports Information about a ZIP file. zip
barcoding_key_csv The master debarcoding key in a CSV file csv
bead_removal Boolean to indicate whether beads have been removed after normalization process < boolean >
control_files_analysis Information about a ZIP file. zip
date_of_acquisition Date of CyTOF batch acquisition. < string >
debarcoding_protocol The strategy/kit used to barcode CyTOF samples, if any < string >
harware_version Version no. of instrument hardware used. < string >
injector Name of Injector component used as part of the CyTOF software < string >
instrument Name of CyTOF instrument on which experiment was conducted. < string >
normalization_method Process used for normalization prior to analysis, if any. < string >
  • "Fluidigm"
  • "Normalizer v0.3"
  • "Other"
source_fcs Information about a FCS file. fcs
spike_in_fcs Processed FCS file for the spike-in. fcs

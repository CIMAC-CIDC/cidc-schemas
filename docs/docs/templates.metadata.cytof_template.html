CIDC Data Model

Metadata file for CYTOF

Description: Metadata information for mass-cytometry (CYTOF).

Worksheet - Acquisition and Preprocessing

Preamble Rows

Property Description Data Type
assay run id User defined unique identifier for this assay run. Not necessarily unique across batches. assays.cytof_assay_core#assay_run_id
batch id CyTOF batch identification number. Unique to every batch. assays.cytof_assay_core#batch_id
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
instrument Name of CyTOF instrument on which experiment was conducted. assays.cytof_assay#instrument
injector Name of Injector component used as part of the CyTOF software assays.cytof_assay#injector
date of acquisition Date of CyTOF batch acquisition. assays.cytof_assay#date_of_acquisition
acquisition buffer Cell staining buffer used for antibody and cell dilution. assays.cytof_assay#acquisition_buffer
bead removal Boolean to indicate whether beads have been removed after normalization process assays.cytof_assay#bead_removal
normalization method Process used for normalization prior to analysis, if any. assays.cytof_assay#normalization_method
debarcoding protocol The strategy/kit used to barcode CyTOF samples, if any assays.cytof_assay#debarcoding_protocol
harware version Version no. of instrument hardware used. assays.cytof_assay#harware_version
concatenated spike in fcs filename Path to a file on a User's computer. This file contains the normalized & debarcoded spike-in for a batch. assays.components.local_file#file_path
concatenated raw fcs filename(s) Path to a file on a user's computer. assays.components.local_file#file_path

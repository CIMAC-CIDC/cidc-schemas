RNA Gene Expression Assay

Description: Additional properties to describe a RNA assay for expression analysis.

Properties

Property Description Data Type
Inherits all properties from schema: assays.components.assay_core
Inherits all properties from schema: assays.components.ngs_assay_core
enrichment_method
required 		Method used for enriching for mRNA. < string >
  • "PolyA capture"
  • "Transcriptome capture"
  • "Ribo minus"

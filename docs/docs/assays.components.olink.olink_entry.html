CIDC Data Model

Olink assay record

Description: A single data record from Olink assay.

Properties

Property Description Data Type
number_of_samples
required 		Number of samples analyzed in the assay. < number >
npx_manager_version
required 		Version of NPX manager used for the analysis. < string >
files
required 		Olink assay input files. assays.components.olink.olink_input
run_date Date when the Olink assay was run. < string : date >
run_time Time when the Olink assay was run. < string : time >
instrument Instrument used for the Olink assay. < string >
fludigm_application_version Version of the Fluidgm application used. < string >
fludigm_application_build Build of the Fluidgm application used. < string >
chip_barcode Chip barcode number. < string >
probe_type Probe Type. < string >
passive_reference Passive Reference. < string >
quality_threshold Threshold used for quality cutoff. < number >
baseline_correction Method used for baseline correction. < string >
number_of_samples_failed Number of samples failed in the assay. < number >

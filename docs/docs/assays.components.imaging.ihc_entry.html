CIDC Data Model

IHC assay record

Description: A single data record from IHC assay.

Properties

Property Description Data Type
percentage_tumor_positive
required 		Percentage of tumor positive (0 - 100). < number >
tumor_positive_intensity
required 		Intensity of positive tumor (1, 2, or 3). < integer >
average_tumor_marker_intensity
required 		Average intensity of tumor marker (0 - 3). < number >
percent_inflammation_marker_positive
required 		Percentage of positive marker inflammation (0 - 100). < integer >
average_inflammation_marker_intensity
required 		Average intensity of marker inflammation (0 - 3). < number >
clinically_positive
required 		Indicates whether the marker is considered positive by FDA guidelines. < integer >
percentage_viable_tissue
required 		Percentage of tissue that is viable (0 - 100). < integer >
percentage_tumor
required 		Percentage of tumor (0 - 100). < integer >
degree_lymphoid_infiltrate
required 		Degree of lymphoid infiltrate (0, 1, 2, or 3). < integer >
percentage_fibrosis
required 		Percentage of fibrosis (0 - 100). < integer >
files
required 		IHC assay input/output files. assays.components.imaging.ihc_input

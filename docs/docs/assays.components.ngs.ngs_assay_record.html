CIDC Data Model

NGS assay record

Description: A single data record from an NGS assay

Properties

Property Description Data Type
cimac_participant_id
required 		Participant identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-1234. participant#cimac_participant_id
cimac_sample_id
required 		Specimen identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. sample#cimac_sample_id
cimac_aliquot_id
required 		Aliquot identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. aliquot#cimac_aliquot_id
files
required 		Paired end sequencing fastq file pair and read group mapping file assays.components.fastq_pair_and_mapping
library_kit_lot
required 		Lot number for the library construction kit. < string >
library_prep_date
required 		Date of library construction. < string : date >
input_ng Amount of DNA/RNA (in ng) used for library construction. < integer >
library_yield_ng Resulting yield (in ng) from library construction. < integer >
average_insert_size The average insert size for the library. < integer >

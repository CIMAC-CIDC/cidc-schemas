TCRseq Assay

Description: Additional properties to describe a TCR assay for expression analysis.

Property Description Data Type
assay_creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
batch_id TCRseq batch identification number. < string >
protocol_identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
sample_sheet A Comma-Separated Value (CSV) text file describing the samples in this batch. csv
sequencer_platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
sequencing_run_date Date when the TCR sequencing was run. < string : date >

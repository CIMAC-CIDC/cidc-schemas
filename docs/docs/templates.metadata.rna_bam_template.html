CIDC Data Model

Metadata file for RNA expression [BAM]

Description: Metadata information for rna expression data submission via BAM files.

Worksheet - RNA

Preamble Rows

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
enrichment method Method used for enriching for mRNA. assays.rna_assay#enrichment_method
enrichment kit Vendor for the bait set used for enrichment, e.g. Twist, Agilent, IDT. assays.rna_assay#enrichment_vendor_kit
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. assays.components.ngs_assay_core#sequencer_platform
paired end reads Indicates if the sequencing was performed paired or single ended. assays.components.ngs_assay_core#paired_end_reads

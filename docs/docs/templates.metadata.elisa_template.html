Worksheet - ELISA

Preamble Rows

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
run id User defined unique identifier for this assay run. assays.elisa_assay#assay_run_id
serology file XLSX ELISA results file name on a user's computer. assays.components.local_file#file_path

Property Description Data Type
target Antibody type collected for this study. assays.components.antibody#antibody
clone Resulting type of antibody clone from primary antibody sample. assays.components.antibody#clone
antibody company Company from which antibody sample was derived. assays.components.antibody#company
catalog number Antibody registry catalog number assigned to antibody sample. assays.components.antibody#cat_num
lot number Identification number assigned to the particular quantity or lot of material from manufacturer. assays.components.antibody#lot_num
isotope Antibody isotype used to help differentiate antibody signals. assays.components.elisa.elisa_antibody#isotope

stain_type Type of staining method used for antibody. assays.components.elisa.elisa_antibody#stain_type
usage How this antibody should be used in automatic analysis assays.components.elisa.elisa_antibody#usage

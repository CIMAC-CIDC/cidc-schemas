CIDC Documentation

Image Artifact

Description: Information about an image file.

Properties

Property Description Data Type
height Height of the image. < integer >
width Width of the image. < integer >
channels Number of channels in the image. < integer >
data_format Data format. < const: IMAGE >
upload_placeholder A placeholder for when artifact file is being uploaded. core#upload placeholder
artifact_creator The name of the center that created this artifact. core#artifact creator
uploader The name of the person uploading the artifact. core#uploader
uuid UUID of artifact. core#uuid
file_name The name of the file with extension. core#file name
object_url URL to artifact within Google Bucket. core#object url
uploaded_timestamp Timestamp of when artifact was loaded into the system. core#uploaded timestamp
file_size_bytes File size in bytes. core#file size bytes
md5_hash MD5 Hash of artifact. Not available for composite GCS objects. core#md5 hash
crc32c_hash CRC32c Hash of artifact. core#crc32c hash
visible Indicates if the artifact is visible. If set to false, the artifact is effectively deleted. core#visible
artifact_category Artifact category. core#artifact category
facet_group The internal data category for this artifact core#facet group

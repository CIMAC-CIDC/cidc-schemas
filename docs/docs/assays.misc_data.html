Miscellaneous data

Description: Miscellaneous data not supported directly by any other assay.

Property Description Data Type
assay_creator
required 		Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
protocol_identifier
required 		Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >

name

file_description An optional long string describing the file, its purpose, and/or its content. < string >

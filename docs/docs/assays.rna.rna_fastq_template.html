CIDC Documentation

RNA expression [FASTQ] assay template

Description: RNA expression data submission via FASTQ files.

Property Description Data Type
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
assay creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
enrichment method Method used for enriching for mRNA. < string >
  • "PolyA capture"
  • "Transcriptome capture v1"
  • "Ribo minus"
enrichment kit Vendor for the bait set used for enrichment, e.g. Twist, Agilent, IDT. < string >
  • "Agilent"
  • "Twist"
  • "IDT"
  • "NEB"
  • "Illumina - TruSeq Stranded PolyA mRNA"
  • "Illumina - TruSeq Stranded mRNA Sample Prep Kit"
  • "Illumina- TruSeq RNA Access Library Prep Kit"
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
paired end reads Indicates if the sequencing was performed paired or single ended. < string >
  • "Paired"
  • "Single"

Samples
Property Description Data Type
cimac id Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. < string > as regex
^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
forward fastq Fastq file for the first fragment. fastq.gz
reverse fastq Fastq file for the second fragment. fastq.gz
library yield ng Resulting yield (in ng) from library construction. < number >
dv200 The percentage of fragments > 200 nucleotides. (0-1) < number >
rqs RNA quality score. (0-10) < number >
rin RNA integrity number. (0-10) < number >
quality flag Flag used for quality. < number >

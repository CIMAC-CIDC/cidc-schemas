CIDC Documentation

Aliquot

Description: Pertaining to a portion (volume or weight) of the whole.

Properties

Property Description Data Type
slide_number
required 		Two digit number that indicates the sequential order of slide cuts, assigned by the CIMAC-CIDC Network. < string > as regex
^[0-9]{1,2}$
aliquot_replacement
required 		Status of aliquot if replacement is/was requested. < string >
  • "Replacement Requested"
  • "Replacement Tested"
  • "N/A"
aliquot_status
required 		Status of aliquot used for other assay, exhausted, destroyed, or returned. < string >
  • "Aliquot Returned"
  • "Aliquot Exhausted"
  • "Remainder used for other Assay"
  • "Other"
  • "Aliquot Leftover"
quantity Quantity of each aliquot shipped. < integer >
material_extracted The type of biological material that was extracted from this aliquot. < string >
  • "DNA"
  • "RNA"
  • "cfDNA"
  • "Other"
extracted_concentration The concentration of biological material that was extracted from this aliquot. < string >
aliquot_amount The amount of extracted aliquot shipped. Example: 400 ng < string >
lymphocyte_influx Extent of lymphocytic infiltration into the tumor stroma or surrounding environment. Example: 2 < string >

CIDC Data Model

NGS Sequencing Artifact

Description: Additional properties to describe a NGS Sequencing Artifact.

Each NGS Sequencing Artifact is linked to:

  • exactly_one artifact

Property Description Data Type Required/Optional
sequencer_platform Sequencing Platform, eg. Illumina, IonTorrent <string>
required
sequencer_model Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. <string>
required
library_kit_vendor Vendor for the library construction kit, e.g. KAPA <string>
required
library_kit_version The version of the library construction kit used, e.g. Hyper Prep <string>
required
library_kit_lot Lot number for the library construction kit <string>
required
paired_end_reads Indicate if the sequencing was performed paired or single ended <boolean>
required
read_length Number of cycles for each sequencing read <integer>
required
library_prep_date Date of library construction. <string> date
required
dna_input_ng Amount of DNA (in ng) used for library construction <integer>
required
library_yield_ng Resulting yield (in ng) from library construction <integer>
required
average_insert_size The average insert size for the library <integer>
required
forward_fastq Fastq file for the forward read <string>
required
reverse_fastq Fastq file for the reverse read <string>
required
read_group_mapping_file Stores read group information for each read in the fastq files. Needed for when samples are run across multiple lanes. <string>
required

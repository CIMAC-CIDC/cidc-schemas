mIF antibody

Description: Data specific to antibody preparation on the multiplex immunofluorescence platform.

Properties

Property Description Data Type
Inherits all properties from schema: assays.components.antibody

required 		Wavelength registered for Alexa Fluor processing. < integer >
primary_ab_dilution
required 		Concentration ratio for primary antibody dilution. < string >
dilutent
required 		Dilution agent used for antibody sample. < string >
fluor_dilution
required 		Concentration ratio for fluor dilution agent. < string >
antigen_retrieval_time
required 		Length of time needed for antigen retrieval. < string : time >
primary_incubation_time
required 		Length of time for primary antibody incubation. < string : time >
amplification_time
required 		Length of time required for amplified signal detection. < string : time >
antibody Antibody type collected for this study. assays.components.antibody#antibody
clone Resulting type of antibody clone from primary antibody sample. assays.components.antibody#clone
company Company from which antibody sample was derived. assays.components.antibody#company
cat_num Antibody registry catalog number assigned to antibody sample. assays.components.antibody#cat_num
lot_num Identification number assigned to the particular quantity or lot of material from manufacturer. assays.components.antibody#lot_num
export_name Name of this antibody in InForm export, e.g. 'CD8 (Opal 520)' < string >
staining_order Order of staining applied < integer >

CIDC Data Model

CyTOF assay record

Description: A single data record from CyTOF assay batch

Properties

Property Description Data Type
processed_fcs_filename
required 		Filename of a processed CyTOF FCS file. < string >
source_fcs_filenames
required 		Filename of source CyTOF FCS file. < string >
files
required 		assays.components.cytof.cytof_input
concatenation_version Concatenation of FCS files into a single FCS version number. < string >
normalization_version Version of normalization for CyTOF assay batch. < string >
beads_removed Confirmation of beads being removed after normalization process. < string >
  • Y
  • N
staining_date Date of staining for CyTOF assay slides. < string : date >
batch_id Identification number for CyTOF assay batch. < string >
debarcoding_protocol Protocol for the download and installation of the Debarcoder CyTOF software feature. < string >
debarcoding_key Key required for access to the Debarcoding software. < string >

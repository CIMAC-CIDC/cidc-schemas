CIDC Documentation

TCRseq Assay

Description: Additional properties to describe a TCR assay for expression analysis.

Property Description Data Type
batch_id
required 		TCRseq batch identification number. < string >
assay_creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
sample_sheet A Comma-Separated Value (CSV) text file describing the samples in this batch. artifacts.artifact_csv
sequencer_platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
sequencing_run_date Date when the TCR sequencing was run. < string : date >

Property Description Data Type
records > cimac_id Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. < string > as regex
^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
records > files > replicates > replicate_id Replicate identifier. < string >
records > files > replicates > i1 Fastq file for the index of Read 1. artifacts.artifact_fastq_gz
records > files > replicates > i2 Fastq file for the index of Read 2. artifacts.artifact_fastq_gz
records > files > replicates > r1 Fastq file for the first fragment. artifacts.artifact_fastq_gz
records > files > replicates > r2 Fastq file for the second fragment. artifacts.artifact_fastq_gz
records > files > replicates > rna_quantity_ng RNA quantity (in ng). < number >
records > tsv_file Information about a tab Separated file. artifacts.artifact_tsv
controls > id A unique ID for this control. < string >
controls > tsv_file Information about a tab Separated file. artifacts.artifact_tsv

Property Description Data Type
batch_id
required 		TCRseq batch identification number. < string >
report_trial Information about a GZ file. artifacts.artifact_gz
summary_info Information about a comma separated file. artifacts.artifact_csv

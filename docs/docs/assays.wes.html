CIDC Documentation

WES Assay

Description: Information about the acquisition and analysis of Whole Exome Sequencing data.

Property Description Data Type
bait_set
required 		Bait set ID. < string >
  • "whole_exome_illumina_coding_v1"
  • "broad_custom_exome_v1"
  • "TWIST Dana Farber Custom Panel"
  • "TWIST Custom Panel PN 101042"
  • "Agilent SS Human All Exon V4"
protocol_identifier
required 		Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. < string >
read_length
required 		Number of cycles for each sequencing read. < integer >
assay_creator Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
  • "Broad"
library_kit The library construction kit. < string >
  • "Hyper Prep ICE Exome Express: 1.0"
  • "KAPA HyperPrep"
  • "IDT duplex UMI adapters"
  • "KAPA HyperPrep with IDT duplex UMI adapters"
  • "TWIST"
paired_end_reads Indicates if the sequencing was performed paired or single ended. < string >
  • "Paired"
  • "Single"
sequencer_platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
sequencing_protocol Protocol and version used for the sequencing. < string >
  • "Express Somatic Human WES (Deep Coverage) v1.1"
  • "Somatic Human WES v6"
  • "TWIST Custom Panel PN 101042"
  • "TWIST"

Property Description Data Type
required
type: array

Property Description Data Type
cimac_id
required 		Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. < string > as regex
^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
quality_flag Flag used for quality. < number >
sequencing_date Date of sequencing. < string : date >
type: object

Property Description Data Type
r1 Fastq file for the first fragment. fastq gz
r2 Fastq file for the second fragment. fastq gz
bam A BAM file, aligned or not, which contains both pairs of sequencing libraries bam

Entity Description
WES [BAM] assay template Whole Exome Sequencing (WES) data submission via BAM files.
WES [FASTQ] assay template Whole Exome Sequencing (WES) data submission via FASTQ files.

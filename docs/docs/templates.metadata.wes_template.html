CIDC Data Model

Metadata file for WES

Description: Metadata information for whole exome sequencing (WES) analysis.

Worksheet - WES

Preamble Rows

Property Description Data Type
lead organization study id Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#lead_organization_study_id
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
enrichment kit Vendor for the bait set used for enrichment , e.g. Twist, Agilent, IDT. assays.wes_assay#enrichment_vendor_kit
library kit vendor Vendor for the library construction kit, e.g. KAPA. assays.components.ngs_assay_core#library_vendor_kit
sequencer platform Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. assays.components.ngs_assay_core#sequencer_platform
paired end reads Indicates if the sequencing was performed paired or single ended. assays.components.ngs_assay_core#paired_end_reads
read length Number of cycles for each sequencing read. assays.components.ngs_assay_core#read_length

Data Columns

Samples
Property Description Data Type
cimac participant id Participant identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-1234. participant#cimac_participant_id
cimac sample id Specimen identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. sample#cimac_sample_id
cimac aliquot id Aliquot identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. aliquot#cimac_aliquot_id
library kit lot Lot number for the library construction kit. assays.components.ngs.ngs_assay_record#library_kit_lot
enrichment lot Lot number for the bait set. assays.components.ngs.wes_assay_record#enrichment_vendor_lot
library prep date Date of library construction. assays.components.ngs.ngs_assay_record#library_prep_date
capture date Date of the hybrid capture enrichment. assays.components.ngs.wes_assay_record#capture_date
dna input ng Amount of DNA/RNA (in ng) used for library construction. assays.components.ngs.ngs_assay_record#input_ng
library yield ng Resulting yield (in ng) from library construction. assays.components.ngs.ngs_assay_record#library_yield_ng
average insert size The average insert size for the library. assays.components.ngs.ngs_assay_record#average_insert_size
forward fastq Path to a file on a user's computer. assays.components.local_file#file_path
reverse fastq Path to a file on a user's computer. assays.components.local_file#file_path
read group mapping file Path to a file on a user's computer. assays.components.local_file#file_path

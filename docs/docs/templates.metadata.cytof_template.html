CIDC Data Model

Metadata file for CYTOF

Description: Metadata information for mass-cytometry (CYTOF).

Worksheet - Acquisition and Preprocessing

Preamble Rows

Property Description Data Type
assay run id User defined unique identifier for this assay run. assays.cytof_assay#assay_run_id
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay creator Indicates what site is filling out the assay. assays.components.assay_core#assay_creator
panel name Standardized CyTOF panel name used for antibody sample development. assays.cytof_assay#panel_name
instrument Name of CyTOF instrument on which experiment was conducted. assays.cytof_assay#instrument

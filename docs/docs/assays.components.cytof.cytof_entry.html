Properties

Property Description Data Type

required 		Aliquot identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. aliquot#cimac_aliquot_id
files
required 		CyTOF Assay Input Files assays.components.cytof.cytof_input
cimac_participant_id Participant identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-1234. participant#cimac_participant_id
cimac_sample_id Specimen identifier assigned by the CIMAC-CIDC Network. Example: CIMAC-12453. sample#cimac_sample_id
panel_name Standardized CyTOF panel name used for antibody sample development. < string >
batch_id CyTOF batch identification number. < string >
date_of_acquisition Date of CyTOF batch acquisition. < string >
instrument Name of CyTOF instrument on which experiment was conducted. < string >
injector Name of Injector component used as part of the CyTOF software < string >
acquisition_buffer Cell staining buffer used for antibody and cell dilution. < string >
average_event_per_second Analysis rate of CyTOF expressions per second. < number >
run_time Length of run time for CyTOF experiment. < string >
notes Any notes pertaining to CyTOF acquisition. < string >
concatenation_version Concatenation of FCS files into a single FCS version number. < string >
normalization_version Version of normalization for CyTOF assay batch. < string >
beads_removed Confirmation of beads being removed after normalization process. < string >
  • "Y"
  • "N"
staining_date Date of staining for CyTOF assay slides. < string : date >
debarcoding_protocol Protocol for the download and installation of the Debarcoder CyTOF software feature. < string >
debarcoding_key Key required for access to the Debarcoding software. < string >
preprocessing_notes Any notes pertaining to preprocessing of CyTOF data. < string >

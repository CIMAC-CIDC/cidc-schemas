RNAseq Gene Expression Assay

Description: An RNA assay for expression analysis.

Property Description Data Type
assay_creator
required 		Indicates what site is filling out the assay. < string >
  • "DFCI"
  • "Mount Sinai"
  • "Stanford"
  • "MD Anderson"
  • "Adaptive"
enrichment_method
required 		Method used for enriching for mRNA. < string >
  • "PolyA capture"
  • "Transcriptome capture v1"
  • "Ribo minus"
sequencer_platform
required 		Sequencer Model, e.g. HiSeq 2500, NextSeq, NovaSeq. < string >
  • "Adaptive"
  • "Illumina - HiSeq 2500"
  • "Illumina - HiSeq 3000"
  • "Illumina - NextSeq 550"
  • "Illumina - HiSeq 4000"
  • "Illumina - NovaSeq 6000"
  • "MiSeq"
enrichment_vendor_kit Vendor for the bait set used for enrichment, e.g. Twist, Agilent, IDT. < string >
  • "Agilent"
  • "Twist"
  • "IDT"
  • "NEB"
  • "Illumina - TruSeq Stranded PolyA mRNA"
  • "Illumina - TruSeq Stranded mRNA Sample Prep Kit"
  • "Illumina- TruSeq RNA Access Library Prep Kit"
paired_end_reads Indicates if the sequencing was performed paired or single ended. < string >
  • "Paired"
  • "Single"

Property Description Data Type
cimac_id Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as CTTTPPPSS.AA for trial code TTT, participant PPP, sample SS, and aliquot AA. < string > as regex
^C[A-Z0-9]{3}[A-Z0-9]{3}[A-Z0-9]{2}.[0-9]{2}$
dv200 The percentage of fragments > 200 nucleotides. (0-1) < number >
bam A BAM file, aligned or not, which contains both pairs of sequencing libraries artifacts.artifact_bam
r1 Fastq file for the first fragment. artifacts.artifact_fastq_gz
r2 Fastq file for the second fragment. artifacts.artifact_fastq_gz
library_yield_ng Resulting yield (in ng) from library construction. < number >
quality_flag Flag used for quality. < number >
rin RNA integrity number. (0-10) < number >
rqs RNA quality score. (0-10) < number >

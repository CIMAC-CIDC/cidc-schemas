CIDC Data Model

Shipping manifest for extracted normal DNA samples

Description: Shipping / receiving manifest for transferring extracted normal DNA samples.

Worksheet - Shipment

Preamble Rows

Property Description Data Type
manifest id Filename of the manifest used to ship this sample. Example: E4412_PBMC. shipping_core#manifest_id
protocol identifier Trial identifier used by lead organization, ie. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412. clinical_trial#protocol_identifier
assay priority Priority of the assay as it appears on the intake form. shipping_core#assay_priority
assay type Assay and sample type used. shipping_core#assay_type
receiving party Site where sample was shipped to be assayed. shipping_core#receiving_party
courier Courier utilized for shipment. shipping_core#courier
tracking number Air bill number assigned to shipment. Example: 4567788343. shipping_core#tracking_number
account number Courier account number to pay for shipping if available. Example: 45465732. shipping_core#account_number
shipping condition Type of shipment made. shipping_core#shipping_condition
date shipped Date of shipment. shipping_core#date_shipped
date received Date of receipt. shipping_core#date_received
quality of shipment Indication that specimens were received in good condition. shipping_core#quality_of_shipment
ship from Contact information for shipment. shipping_core#ship_from
ship to Physical shipping address of the destination. shipping_core#ship_to

Worksheet - Samples

Data Columns

IDs
Property Description Data Type
entry (#) Provides a numbered identifier for patient (sample) entry in a shipment manifest. sample#shipping_entry_number
collection event name Categorical description of timepoint at which the sample was taken. sample#collection_event_name
cohort name Categorical description of cohorts, arms, and treatment groups. participant#cohort_name
participant ID Trial Participant Identifier. participant#participant_id
parent sample ID Sample identifier assigned by the biorepository site. Example: EA123-45. sample#parent_sample_id
processed sample ID Aliquot identifier assigned by the biorepository site: EA123-45-1 sample#processed_sample_id
CIMAC ID Specimen identifier assigned by the CIMAC-CIDC Network. Formatted as C????????.?? sample#cimac_id
Specimen Info Filled by Biorepository
Property Description Data Type
box number Identifier if sample shipment container includes multiple boxes for each assay. sample#box_number
sample location Sample location within the shipping container. Example: A1. sample#sample_location
type of sample Type of sample sent. sample#type_of_sample
sample collection procedure Indicates the specimen source of the sample shipped. Example: Na Heparin blood draw aliquots (2 of three), FFPE block #52 sample#sample_collection_procedure
type of primary container The format in which the sample was sent. sample#type_of_primary_container
processed sample type The type of processing that was performed on the collected specimen by the Biobank for storage. sample#processed_sample_type
processed sample volume Volume of the processed sample. sample#processed_sample_volume
processed sample volume units Volume units of each processed sample. sample#processed_sample_volume_units
processed sample quantity Quantity of the processed sample (e.g. number of slides cut for DNA extraction). sample#processed_sample_quantity
processed sample derivative The type of derivative or analyte extracted from the specimen to be shipped for testing. sample#processed_sample_derivative
sample derivative volume Volume of the analyte or derivative shipped. sample#sample_derivative_volume
sample derivative volume units Volume units of each analyte or derivative shipped. sample#sample_derivative_volume_units
sample derivative concentration The concentration of analyte or derivative shipped. sample#sample_derivative_concentration
sample derivative concentration units The concentration units for the analyte or derivative shipped. sample#sample_derivative_concentration_units
DNA quality
Property Description Data Type
din Provides a DNA Integrity Number as an indication of extraction quality (values of 1-10) sample#din
a260/a280 Provides an absorbance percentage ratio indicating purity of DNA (values of 0 to 2) sample#a260_a280
a260/a230 Provides an absorbance percentage ratio indicating presence of contaminants (values of 0 to 3) sample#a260_a230
Filled by CIMAC Lab
Property Description Data Type
material used Receiving site indicates how much material was used for assay purposes. sample#material_used
material used units Units for the amount of material used; should be the same value as Specimen Analyte units. sample#material_used_units
material remaining Receiving site indicates how much material remains after assay use. sample#material_remaining
material remaining units Units for the amount of material remaining. sample#material_remaining_units
material storage condition Storage condition of the material once it was received. sample#material_storage_condition
quality of sample Final status of sample after QC and pathology review. sample#quality_of_sample
sample replacement Indication if sample replacement is/was requested. sample#sample_replacement
residual sample use Indication if sample was sent to another location or returned back to biorepository. sample#residual_sample_use
comments Comments on sample testing. sample#comments

CIDC Data Model

Shipping manifest for PBMC samples

Description: Shipping / receiving manifest for transferring PBMC samples.

Worksheet - PBMCs

Preamble Rows

Property Description Data Type
lead organization study id
Trial identifier used by lead organization, e.g. Center for Experimental Therapeutics Program (CTEP) ID or Industry Sponsored ID. This is usually a short identifier. Example: E4412.
 clinical_trial#lead_organization_study_id
manifest id
Filename of the manifest used to ship this sample. Example: E4412_PBMC.
shipping_core#manifest_id
request
Assigns a request identifier (mandatory or optional) for the assay.
 shipping_core#request
assay priority
Priority of the assay as it appears on the intake form.
 shipping_core#assay_priority
assay type
Assay and sample type used. Example: Green-top tube - Plasma (Olink).
 shipping_core#assay_type
batch number
The number corresponding to the batch sent for this assay.
 shipping_core#batch_number
box number
Box identifier for samples, if sample shipment container includes multiple boxes for each sample/assay type.
 shipping_core#box_number
assay run number
Assay run number for samples received.
 shipping_core#assay_run_number
courier
Courier utilized for shipment.
 shipping_core#courier
tracking number
Air bill number assigned to shipment. Example: 4567788343.
 shipping_core#tracking_number
account number
Courier account number to pay for shipping if avaialable. Example: 45465732.
shipping_core#account_number
shipping condition
Type of shippment made.
 shipping_core#shipping_condition
date shipped
Date of shipment.
 shipping_core#date_shipped
num of samples shipped
Count of specimens distributed by site.
 shipping_core#num_of_samples_shipped
num of samples good
Number of Samples Received in Good Condition.
 shipping_core#num_of_samples_good
num of path report
Number if pathology report received.
 shipping_core#num_of_path_report
ship from
Shipping from [free text: person or organization, address, phone number, email address.]
 shipping_core#ship_from
ship to
Shipping to [free text: person or organization, address, phone number, email address.]
 shipping_core#ship_to
